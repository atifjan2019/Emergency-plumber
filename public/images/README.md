# Site images

Drop image files here using the paths below and they will appear on the live site.

## Expected files

| Path                                      | Where it appears                  | Recommended size |
| ----------------------------------------- | --------------------------------- | ---------------- |
| `/images/hero-engineer.jpg`               | Home page hero (right column)     | 1200x900         |
| `/images/hero-van.jpg`                    | Home page secondary hero band     | 1600x900         |
| `/images/about-team.jpg`                  | About page                        | 1600x900         |
| `/images/services/burst-pipe.jpg`         | Service card / service page hero  | 1200x800         |
| `/images/services/boiler-repair.jpg`      | Service card / service page hero  | 1200x800         |
| `/images/services/blocked-drain.jpg`      | Service card / service page hero  | 1200x800         |
| `/images/services/leak-detection.jpg`     | Service card / service page hero  | 1200x800         |
| `/images/services/no-hot-water.jpg`       | Service card / service page hero  | 1200x800         |
| `/images/services/heating-repair.jpg`     | Service card / service page hero  | 1200x800         |
| `/images/services/blocked-toilet.jpg`     | Service card / service page hero  | 1200x800         |
| `/images/services/24-7-plumber.jpg`       | Service card / service page hero  | 1200x800         |
| `/images/cities/<slug>.jpg`               | Each city page header             | 1600x900         |

`<slug>` is the city slug used in URLs: `london`, `manchester`, `birmingham`, `glasgow`, `leeds`, `liverpool`, `bristol`, `sheffield`, `edinburgh`, `newcastle`, `nottingham`, `cardiff`.

Until a file exists at the path, the page renders a styled placeholder block in its place.

## Logo and favicon

These are managed in the admin (`/admin/settings`) and stored in Supabase Storage, NOT here.
